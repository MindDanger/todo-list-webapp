# Todo List App
Đây là ứng dụng Todo List đơn giản được xây dựng bằng HTML, CSS và JavaScript
Project này được thực hiện để luyện tập DOM manipulation và JavaScript cơ bản.

## Demo

Link demo: https://minddanger.github.io/todo-list-webapp/

## Chức năng

* Thêm task mới
* Đánh dấu task đã hoàn thành khi ấn vào task đó
* Xóa công việc
* Có thể nhấn **Enter** để thêm task sau khi nhập

## Công nghệ sử dụng

* HTML
* CSS
* JavaScript

## Cấu trúc project
```
todo-list-app
│
├── index.html
├── style.css
└── script.js
```
